We define a function populateVoicesList that retrieves the list of available voices using speechSynthesis.getVoices() and populates an HTML element with the voice options. We add an event listener for the voiceschanged event of the speechSynthesis object. This event is fired when the list of voices becomes available or changes. Inside the event listener, we call populateVoicesList() to populate the voice list initially and update it whenever the voices change. In your HTML, you should have an element with the id "voicesList" where the voice options will be populated, typically a element. When you run this code, it will populate the select element with a list of available voices on the user's device once the voiceschanged event is triggered. Users can then select a voice from the list for text-to-speech conversion.
